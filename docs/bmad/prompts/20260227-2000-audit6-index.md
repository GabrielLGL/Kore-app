## Issues adressées

### Groupe A — Logo & Nav
- `KoreLogo.tsx` : `role="img" aria-label="Logo Kore"` → `aria-hidden="true"` (toujours décoratif, texte "KORE" visible présent dans tous les contextes)
- `HeroSection.tsx` : `<nav>` sans `aria-label` → `aria-label="Navigation principale"` ; lien logo `href="#"` → `href="/"`

### Groupe B — Privacy page
- Skip link manquant (Tab depuis barre d'adresse → aucun lien d'évitement visible)
- `<main>` sans `id="main-content"` (cible skip link)
- `<table>` sans `<caption>` (WCAG 1.3.1)
- Flèche bas de page sans `aria-hidden="true"` (doublon vs lien texte)
- Liens sous-traitants `target="_blank"` sans `aria-label` signalant le nouvel onglet

### Groupe C — 404 Not-Found
- Skip link manquant
- `<main>` sans `id="main-content"`
