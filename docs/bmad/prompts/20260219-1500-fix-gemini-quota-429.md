# Prompt analysé — fix-gemini-quota-429 — 2026-02-19

## Demande originale
Erreur 429 Gemini : "You exceeded your current quota" — `free_tier_requests limit: 0` pour `gemini-2.0-flash`. Le message inclut "retry in 39s".

## Analyse

### Problème identifié
`gemini-2.0-flash` a un quota free tier = 0 sur le compte de l'utilisateur (pas de tier gratuit pour ce modèle, ou quota épuisé). Deux sous-problèmes :
1. Le modèle utilisé (`gemini-2.0-flash`) n'est pas accessible gratuitement → changer pour `gemini-2.0-flash-lite` qui est explicitement sur free tier
2. L'erreur 429 n'a pas de hint spécifique dans `SettingsScreen.tsx` (seul le 403 en a un)

### Fichiers concernés
- `mobile/src/services/ai/geminiProvider.ts` — changer `GEMINI_URL` vers `gemini-2.0-flash-lite`
- `mobile/src/screens/SettingsScreen.tsx` — ajouter hint 429 dans `handleTestConnection`
