# Test manuel Android multi-devices

## Statut : En attente

## Contexte
Avant soumission Play Store, un test manuel approfondi sur plusieurs devices/emulateurs Android est necessaire pour s'assurer que l'app fonctionne correctement partout.

## Actions
1. Tester sur au moins 3 configurations :
   - Emulateur Pixel 7 (API 34) — flagship
   - Emulateur Pixel 4a (API 30) — mid-range
   - Device physique si disponible
2. Checklist par device :
   - [ ] Onboarding complet (3 etapes)
   - [ ] Creation programme + session + exercices
   - [ ] Lancement et completion d'un workout
   - [ ] Timer par exercice
   - [ ] Notes par exercice
   - [ ] Historique et graphiques
   - [ ] Catalogue d'exercices (filtres, recherche, creation)
   - [ ] Superset/Circuit (grouper, degrouper)
   - [ ] Settings (theme toggle, langue FR/EN)
   - [ ] Rappels (notifications)
   - [ ] Navigation complete (20 routes)
   - [ ] Rotation ecran
   - [ ] Mode avion (offline-first)
3. Documenter les bugs trouves
4. Corriger les bugs critiques avant soumission

## Risques
- Bugs specifiques a certaines versions d'Android
- Problemes de layout sur petits ecrans
- Performance sur devices bas de gamme

## Priorite : IMPORTANT
Derniere verification avant soumission.
